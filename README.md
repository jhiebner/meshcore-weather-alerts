# MeshCore Weather Alerts

A Python application that monitors the National Weather Service (NWS) for severe weather alerts and broadcasts them across a MeshCore network.

## Features

- National Weather Service (NWS) integration
- Broadcast alerts over MeshCore
- Interactive setup wizard
- Track up to three counties
- Select which alert types to broadcast
- Duplicate alert suppression
- Configurable rebroadcast intervals
- Linux systemd service support
